About

  • Accessibility Features

    • 1.Tab Control

    You can navigate through the dialing pad using tab key.

    2.Keyboard Shortcuts

    You can access different dialing buttons, switch between different tabs using accesskeys.

    3.Accessible Table

    The Contact List is displayed as an accessible table.

    4.Accessible Images

    The images on the Contact List page come with descriptions even using the alt tag.

    5.ARIA Landmark Roles

    "Main" and "Navigation" roles are added to the app.

    6.ARIA Live Regions

    An "ARIA live region" with the value "polite" is added to each tab div.
